html基础head

发表于 2017-07-02   |   分类于 HTML/CSS   |     |  

[TOC]

DOCTYPE

告知浏览器文档使用哪种HTML或者XHTML

  • html5中doctype不区分大小写

charset

声明文档使用的字符编码
html5中：<meta charset="utf-8">
html5之前：<meta http-equiv="Content-Type" content="text/html" charset="utf-8">

lang

<html lang="zh-cmn-Hans">
使用zh-cmn-Hans比使用zh-CN兼容性更好，zh-CN带有地域性，无法匹配zh-SG（新加坡）

优先使用IE最新版和chrome

content的取值为webkit，ie-comp，ie-stand之一，区分大小写，分别代表用极速模式，兼容模式，IE模式打开。

百度禁止转码

通过百度手机打开网页时，百度可能会对你的网页进行转码，脱下你的衣服，往你的身上贴狗皮膏药的广告，为此可在 head 内添加

SEO

  • 关键词：
  • 页面描述description
  • 定义网页作者author

  • 定义网页搜索引擎索引模式

为设备添加viewport

<meta name="viewport" content="width=device-width,initial-scale=1,max-scale=3,minimum-scale=1,user-scale=no ">

user-scalable 是否允许用户缩放(yes/no)

如果你的网站不是响应式的，请不要使用 initial-scale 或者禁用缩放:<meta name="viewport" content="width=device-width,user-scalable=yes">

适配 iPhone 6 和 iPhone 6plus 则需要写：

rss订阅

<meta rel="alternatite" type="application/rss+xml" title="RSS" href="/rss.xml">

favicon

<link rel="shortcut icon" type="image/ico" href="图标地址" >

关闭chrome浏览器下翻译插件

<meta name=google" value="notranslate">
